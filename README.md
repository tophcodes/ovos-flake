# NixOS flake for OpenVoiceOS

A standalone NixOS flake providing native Nix packages and modules for OpenVoiceOS.

## Features

- **NixOS module** for OpenVoiceOS server (system-wide, network-accessible)
- **Native Nix packages** for all OVOS components (no Docker)
- **Declarative configuration** for voice assistant setup
- **Systemd service** integration
